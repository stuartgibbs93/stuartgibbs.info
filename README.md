# StuartGibbs.info — Custom WordPress Theme

This repository contains the custom WordPress theme built for **StuartGibbs.info**, a personal portfolio site designed to showcase clean front‑end development, thoughtful UI structure, and a modern build workflow.

The theme is built from scratch using WordPress best practices, modular PHP templates, SCSS, ES6 JavaScript, and a small React component bundle compiled into the theme.

---

## ⭐ What This Theme Demonstrates

- **Front‑end development skills** using SCSS, modular JavaScript, and component‑based structure  
- **Modern tooling** (Gulp, npm scripts, linting) for a clean and maintainable workflow  
- **React integration** for interactive UI elements  
- **Organized WordPress architecture** with template parts, reusable components, and structured includes  
- **Production‑ready workflow** — compiled assets are included, so the theme runs on any WordPress environment

This theme reflects how I approach building maintainable, scalable front‑end work inside a CMS environment.

---

## 📁 Project Structure

| Folder | Purpose |
|--------|----------|
| `css/` | Compiled CSS |
| `sass/` | SCSS source files |
| `js/` | Theme JavaScript |
| `images/` | Theme images & SVGs |
| `inc/` | PHP includes (customizer, template tags, etc.) |
| `template-parts/` | Reusable template components |
| `languages/` | Translation files |
| `maps/` | Source maps |
| `react/` | React component + build output |

---

## 📦 Deployment

The theme includes all compiled assets and can be deployed to any WordPress environment without requiring Node or build tools on the server.
